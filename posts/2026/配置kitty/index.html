从 Claude Code 说起

Anthropic 开发的 Claude Code 不仅在 Agentic Coding 领域表现卓著，也在 法律 等领域表现出通用潜力。

Claude Code 是作为 CLI 工具呈现的，正如 Emacs-China 一则讨论 所说，这一形态或与 Emacs 颇有渊源。

Claude Code 官方演讲里说：当时把 Claude Code 做成 CLI 以及提供 SDK 而没有做成 Cursor，就是因为 Anthropic 有人用 Emacs 等编辑器 :joy:。

Emacser 从未停止将好用的工具集成到 Emacs 这个伪装成编辑器的操作系统里。对于 Claude Code 而言，已有 claude-code.el, claude-code-ide.el, agent-shell 等好用的插件。然而，一方面 GUI Emacs 使用 vterm 或 eat 作为内置终端来使用 Claude Code 都存在着或多或少的渲染问题。另一方面，Claude Code 不断演化新特性，个人开发者难以跟上 Anthropic 团队的步调。

因此，个人判断，在终端中使用原生 Claude Code 是最好的选择。相应地，Emacs 也可以从 GUI 环境切换到 TUI 环境下。为此需要选择一个好用的终端。Emacs-China 上关于 kitty 的 讨论 很多，看起来值得一试。

配置 kitty

使用命令 brew install --cask kitty 即可安装 kitty 。安装完毕后可参考 sw.kovidgoyal.net/kitty/conf 进行配置。

使用 awk '!/^\s*(#|$)/' ~/.config/kitty/kitty.conf 命令可以打印 kitty 配置文件中的生效部分。

cursor_shape beam
# jelly cursor animation
cursor_trail 1
copy_on_select yes
# 配合 kkp.el 实现 option 到 meta 的映射
macos_option_as_alt yes
# 支持 TUI Emacs 中 Ctrl-Delete 实现向左删除一个词
map ctrl+backspace send_text all \x1b\x7f
# 支持 C-l 将当前行置顶
map ctrl+l scroll_prompt_to_top
# Modus Operandi Tinted
include current-theme.conf
# 设置主字体为 Sarasa Term SC Nerd
font_family      family="Sarasa Term SC Nerd"
font_size        15.0
bold_font        auto
italic_font      auto
bold_italic_font auto
# 使用 JuliaMono 渲染特殊ui符号
symbol_map U+2500-U+259F,U+25A0-U+25FF,U+2800-U+28FF JuliaMono
enabled_layouts splits, stack
allow_remote_control yes
# 启动 claude code
map ctrl+shift+c launch --location=vsplit --bias=25 zsh -l -c "mc --code"
# 在当前布局 splits 和全屏布局 stack 之间切换
map super+z toggle_layout stack
# 恢复 Emacs set mark
map ctrl+shift+2 send_text all \x00
