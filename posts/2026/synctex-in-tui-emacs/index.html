在 TUI Emacs 中浏览 LaTeX 片段

对于概念验证等用途的日志或 wiki 而言，Emacser 往往会拥抱轻量而有特色的 org 文件。由于 org-mode 的强大功能，即使是 LaTeX 最擅长的数学环境，org-mode 也可以提供一定的支持。

GUI Emacs 可参考 Emacs-China 的讨论 package recommend: math-preview 在 org-mode 中开启 LaTeX 公示预览。

(use-package math-preview)

(use-package org-fragtog
  :config
  (add-hook 'org-mode-hook 'org-fragtog-mode)
  (defalias #'org-latex-preview #'math-preview-at-point)
  (defalias #'org-clear-latex-preview #'math-preview-clear-region))

在 kitty 终端中所打开的 TUI Emacs 则可以借助 kitty-graphics.el 来浏览 LaTeX 公式、算法流程图等。启动预览的命令为 C-c C-x C-v ，取消预览的命令为 C-u C-c C-x C-v 。

在 TUI Emacs 中阅读 PDF 文件

kitty-graphics.el 在安装 ghostscript 后可以借助 DocView 来浏览 PDF，然而 DocView 因存在效率问题而广受诟病。在 GUI Emacs 中, 颇为流行的 pdf-tools 的诞生使命之一便是为了取代 DocView 来阅读 PDF 文件，然而 pdf-tools 在 TUI Emacs 并不能使用。

PDF Tools is, among other things, a replacement of DocView for PDF files. The key difference is that pages are not pre-rendered by, say, ghostscript and stored in the file-system, but rather created on-demand and stored in memory.

Emacs-China 的帖子 通过 Kitty 图形协议在终端 Emacs (emacs -nw) 中显示图片 里，一位道友介绍了 bookokrat 这样的 terminal PDF Reader；类似地，量子位在早年间还有一则轰动的报道: 1700页数学笔记火了！全程敲代码，速度飞快易搜索，硬核小哥教你上手LaTeX+Vim，故事主人公熟练地使用 zathura 配合 vim 在数学课记录 LaTeX 板书。

诚然, zathura + vim + LaTeX 的工具链尽管极客，但学习曲线非常陡峭。笔者在 macOS 平台下更常用的是 Skim 阅读器，并且摸索出一种行之有效的 TUI Emacs synctex 的方式。

brew install --cask skim
  • LaTeX 到 pdf 正向跳转，在 Emacs 添加以下配置即可：
(if (display-graphic-p)
  (setq TeX-view-program-selection '((output-pdf "PDF Tools")))
(progn
  (add-to-list 'TeX-view-program-list
               '("Skim" "/Applications/Skim.app/Contents/SharedSupport/displayline -b %n %o %b"))
  (setq TeX-view-program-selection '((output-pdf "Skim")))))
  • pdf 到 LaTeX 反向跳转，需配置 Skim 同步：
选项 设置
检查文件变化 勾选
自动重新加载 勾选
预设 自定义
命令 ${HOME}/.local/bin/skim-to-emacs
参数 %line "%file"

其中 ${HOME}/.local/bin/skim-to-emacs 脚本内容为：

#!/bin/bash
# Skim reverse sync: jump to source in terminal Emacs and focus kitty
# Arguments: %line %file (passed by Skim)

LINE="$1"
FILE="$2"

# Jump to the file and line in Emacs
/opt/homebrew/bin/emacsclient --no-wait +"$LINE" "$FILE"

# Bring kitty to foreground
osascript -e 'tell application "kitty" to activate'

至此，在编译生成的 PDF 文件感兴趣的某一处按下 Shift+Cmd+左键 ，即可跳转回到 LaTeX 文件的相应位置。
